Oratio

Push-to-Talk Diktieren für macOS. Whisper lokal oder Cloud, optionale LLM-Nachbearbeitung — in jede App, per globalem Tastenkürzel.

Apple Silicon · macOS 14+ · SwiftUI · WhisperKit

Features

Eine kleine, fokussierte Menüleisten-App für schnelles Diktieren.

🎙️

Push-to-Talk & Toggle

Taste halten und sprechen oder einmal tippen = Start, nochmal = Stop. 300 ms Pre-Roll-Puffer gegen verlorene Wortanfänge.

🧠

Zwei STT-Backends

Lokal via WhisperKit (Whisper Large v3 Turbo, Apple Neural Engine) oder beliebige OpenAI-kompatible Cloud-API.

✨

LLM-Nachbearbeitung

Rechtschreibung & Grammatik, Professioneller Stil, Höfliche Umformulierung oder eigener Prompt — editierbar & zurücksetzbar.

🌐

Multi-Provider

Presets für OpenAI, Groq, OpenRouter oder eigenen Host. Transkription und Nachbearbeitung unabhängig konfigurierbar.

🔇

Füllwort-Filter

„ähm", „öh", „hmm" und Varianten werden per Regex entfernt — schnell, offline, ohne LLM-Kosten.

🔐

Keychain-Sicherheit

API-Keys werden im macOS-Schlüsselbund gespeichert — separate Einträge für Transkription & Nachbearbeitung möglich.

State Machine — RecordingState

stateDiagram-v2
    [*] --> idle
    idle --> recording : Hotkey down (PTT)\nor Toggle
    recording --> transcribing : Hotkey up\nor Toggle stop
    recording --> idle : Audio < 0.5 s
    transcribing --> idle : Paste done\nor error
    idle --> idle : Auto-sleep 30 s\nEngine teardown

NSApplicationDelegateAdaptor als Coordinator, SwiftUI-Observation für den State.

Audio-Pfad — Pre-Warm & Pre-Roll

flowchart TD
    MIC["🎤 Microphone"] --> TAP["AVAudioEngine Input Tap"]
    TAP --> HP["1-Pol Hochpass @ 60 Hz"]
    HP --> CONV["Converter → 16 kHz Mono Float32"]
    CONV --> RING["Ringpuffer (300 ms, immer aktiv)"]
    CONV --> CAP["Capture-Buffer (während aktiv)"]
    RING -.->|"auf PTT-Start einmal kopieren"| PREROLL["Pre-Roll in Capture-Buffer"]
    PREROLL --> CAP
    CAP --> OUT["[Float] Samples @ 16 kHz"]

UserDefaults & Keychain — Speicher-Layout

Schlüssel Speicher Zweck
Hotkey KeyboardShortcuts Globales Tastenkürzel (Default: ⌥Leertaste)
activationMode UserDefaults Push-to-Talk oder Toggle
selectedBackend UserDefaults Lokal (WhisperKit) oder Cloud-API
Transkription API-Key Keychain: api-key OpenAI/Groq/OpenRouter-Schlüssel für STT
Korrektur API-Key Keychain: correction-api-key Separater Schlüssel für LLM (optional)
correctionMode UserDefaults off / grammar / professional / polite / custom
correctionPrompt_* UserDefaults Eigene Prompts je Modus (überschreibt Default)
Whisper-Modell ~/Documents/huggingface Lokaler CoreML-Modell-Cache (WhisperKit)
